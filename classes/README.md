# classes/

Tout ce qui concerne les classes de personnage jouables : sorts, stats de classe, rendu
visuel du personnage.

Un seul personnage existe aujourd'hui (`sorcier/`) — le dossier est structuré en sous-dossier
par classe dès maintenant pour que l'ajout d'une future classe jouable n'oblige pas à
redécouper l'existant, juste à ajouter `classes/nouvelle-classe/`.
